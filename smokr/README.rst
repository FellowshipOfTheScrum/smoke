Smoke and I/o testing with any language through unittests


* Free software: MIT license
* Documentation: https://smokr.readthedocs.io.


Features
--------

* Basic smoke testing for:

  * C
  * C++
  * Ruby
  * Java
  * Python
  * POSIX Shell
